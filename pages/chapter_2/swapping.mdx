
### 1. Using a Temporary Variable

```python copy showLineNumbers
# Using a temporary variable
a = 5
b = 10

# Swapping
temp = a
a = b
b = temp

print("After swapping:")
print("a =", a)
print("b =", b)
```

**Output**:
```
After swapping:
a = 10
b = 5
```

### 8. Using functions (custom function)

```python copy showLineNumbers
# Using functions (custom function)
def swap(x, y):
    return y, x

a = 5
b = 10

# Swapping
a, b = swap(a, b)

print("After swapping:")
print("a =", a)
print("b =", b)
```

**Output**:
```
After swapping:
a = 10
b = 5
```
