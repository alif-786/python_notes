### If-Else Scenarios in Diagram Form

#### 1. Simple if-else Statement



#### 2. if-elif-else Statement


```mermaid
graph TD;
    A[Condition 1] -->|True| B[Action if True]
    A -->|False| C[Condition 2]
    C -->|True| D[Action if elif True]
    C -->|False| E[Action if False]
```

#### 3. Nested if-else Statement


```mermaid
graph TD;
```
